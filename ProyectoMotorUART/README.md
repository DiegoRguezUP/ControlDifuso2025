# ProyectoMotorUART

Control de un motor vía UART con procesamiento en PC (MATLAB) y firmware en microcontrolador. Incluye envío/recepción de telemetría, cierre de lazo y registro de datos para análisis.

**Versión de MATLAB probada:** 2025a  
**Repositorio:** parte de `ControlDifuso2025` (carpeta `ProyectoMotorUART`)

## Objetivos
- Implementar comunicación UART robusta entre PC y microcontrolador.
- Ejecutar control (PID, Difuso u otro) y evaluar desempeño.
- Registrar telemetría (posición, velocidad, corriente, etc.) para análisis posterior.
